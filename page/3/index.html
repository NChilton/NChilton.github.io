LAMMPS 中 fix nve 命令

发表于 2018-06-25 更新于 2019-01-19 分类于 Learning

LAMMPS 中fix 1 all nve命令执行 NVE 积分更新体系所有粒子位置和速度，在这过程中保持体系粒子数目、体积、总能量不变。该命令最终产生与微正则系综一致的系统轨迹。

NVE 积分

对于 $n$ 粒子体系，每个粒子质量均为 $m$，在 $t$ 时刻位置分别为 $\vec{r}_1,\vec{r}_2,…,\vec{r}_n$，速度分别为 $\vec{v}_1,\vec{v}_2,…,\vec{v}_n$。则体系动能为

$$E_\text{k}=\frac{1}{2}m\sum_{i=1}^N v_i^2,$$

即体系动能可以表示为每个粒子速度的函数 $E_\text{k}=E_\text{k}(\vec{v}_1,\vec{v}_2,…,\vec{v}_n)$。如果体系可以只用对势描述，则体系的势能可表示为每个粒子坐标的函数，即 $E_\text{p}=E_\text{p}(\vec{r}_1,\vec{r}_2,…,\vec{r}_n)$，则体系总能量为 $E = E_\text{k}+E_\text{p}$。

在 LAMMPS 中输出的热力学信息体积为模拟盒子的大小，一般情况下不会改变。所以 NVE 积分保持 $E$ 不变即可。

射流不稳定性

发表于 2018-06-07 更新于 2019-04-07 分类于 Research

射流不稳定性即 **Plateau–Rayleigh instability**，也被称为 Rayleigh instability，用来解释射流断裂为液滴。射流不稳定性的驱动力为表面张力，其作用是使得液柱表面能最小。以下显示了水、甘油、聚乙二醇水溶液的断裂过程，该过程准确称法是下滴（Dripping），与射流（Jet）还是有所不同，前者流体速度慢，后者速度快。但两种过程分析方法相同。以下分析还是针对射流过程，分析过程来自于 MIT 课件。

重力下射流过程稳态分析

考虑流量为 $Q$、密度为 $\rho$、运动粘度为 $\nu$ 的流体从一半径为 $a$ 孔口向下流出，如下图所示。假设射流进口的雷诺数为 $\text{Re} = Q/(a\nu)$ 足够大，可以忽略粘性。并且假设射流入口速度与半径无关。假设速度 $U(z)$，液柱的半径为 $r(z)$。

由伯努利定理知点 A 和 点 B 满足：

$$\frac{1}{2}\rho U_0^2+\rho gz+P_\text{A}=\frac{1}{2}\rho U^2(z)+P_B,\tag{1}$$

局部曲率可以由两个主曲率 $R_1$ 和 $R_2$ 表示：

$$\nabla\cdot\mathbf{n}=\frac{1}{R_1}+\frac{1}{R_2}\approx\frac{1}{r},\tag{2}$$

点 A（入口）和点 B 的压力（Laplace 力）为：

$$P_\text{A}\approx P_0+\frac{\sigma}{a},\quad P_\text{B}\approx P_0+\frac{\sigma}{r},\tag{3}$$

式 $(3)$ 代入式 $(1)$ 得：

$$\frac{1}{2}\rho U_0^2+ \rho gz+P_0+\frac{\sigma}{a}=\frac{1}{2}\rho U^2(z)+P_0+\frac{\sigma}{r},\tag{4}$$

解得：

$$\frac{U(z)}{U_0}=\left[1+\frac{2}{\text{Fr}}\frac{z}{a}+\frac{2}{\text{We}}(1-\frac{a}{r})\right]^{1/2},\tag{5}$$

其中

$$\text{Fr}=\frac{U_0^2}{ga}=\frac{\text{Intertia}}{\text{Gravity}}=\text{Froude Number},$$

$$\text{We}=\frac{\rho U_0^2a}{\sigma}=\frac{\text{Intertia}}{\text{Curvature}}=\text{Weber Number}.$$

由流量守恒得：

$$Q=2\pi \int_0^r U(z)r(z)\text{d}z=\pi a^2U_0=\pi r^2U(z),\tag{6}$$

因此

$$\frac{r(z)}{a}=\left(\frac{U_0}{U(z)}\right)^{1/2}=\left[1+\frac{2}{\text{Fr}}\frac{z}{a}+\frac{2}{\text{We}}\left(1-\frac{a}{r}\right)\right]^{-1/4},\tag{7}$$

当 $\text{We}\rightarrow\infty$ 时，有：

$$\frac{r(z)}{a}\left(1+\frac{2gz}{U_0^2}\right)^{-1/4},\quad\frac{U(z)}{U_0}=\left(1+\frac{2gz}{U_0^2}\right)^{1/2}.$$

扰动分析

不考虑重力的话，平衡态包含无限长半径为 $R_0$ 的流体柱。内部压力平衡边界处表面张力正应力

$$p_0=\sigma \nabla\cdot\mathbf{n}\Rightarrow p_0=\frac{\sigma}{R_0}.\tag{8}$$

考虑小扰动，如上图所示。扰动表面假设如下，$\epsilon$ 为扰动振幅（$\epsilon\ll R_0$）：

$$\widetilde{R}=R_0+\epsilon \mathrm{e}^{\omega t+\mathrm{i}kz},\tag{9}$$

$\omega$ 为不稳定得增长率，频率，可为复数，$k$ 是增长方向上得波数，相应得波长为 $2\pi/k$。

N-S 方程为：

$$\frac{\partial \mathbf{u}}{\partial t}+(\mathbf{u}\cdot\nabla)\mathbf{u}=-\frac{1}{\rho}\nabla p+\nu\Delta\mathbf{u},$$

假设 $\widetilde{u}_r$ 为扰动速度得径向分量，$\widetilde{u}_z$ 为轴向分量，$\widetilde{p}$ 是扰动压力。将扰动量代入 N-S 方程得：

$$\frac{\partial \widetilde{u}_r}{\partial t}=-\frac{1}{\rho}\frac{\partial \widetilde p}{\partial r},\quad\frac{\partial \widetilde{u}_z}{\partial t}=-\frac{1}{\rho}\frac{\partial \widetilde p}{\partial z},\tag{10}$$

线性化连续性方程：

$$\frac{\partial \widetilde{u}_r}{\partial r}+\frac{\widetilde{u}_r}{r}+\widetilde{u}_z=0.$$

假设速度、压力与表面有相同的形状的扰动：

$$\widetilde{u}_r=R(r)\mathrm{e}^{\omega t+\mathrm{i} kz},\quad\widetilde{u}_z=Z(r)\mathrm{e}^{\omega t+\mathrm{i} kz},\quad\widetilde{p}=P(r)\mathrm{e}^{\omega t+\mathrm{i} kz},\tag{11}$$

将上式代入式 $(10)$ 得：

动量方程：

$$\omega R=-\frac{1}{\rho} \frac{\mathrm{d}P}{\mathrm{d} r},\quad \omega Z=-\frac{\mathrm{i}k}{\rho}P,\tag{12}$$

连续性方程：

$$\frac{\mathrm{d} R}{\mathrm{d} r}+\frac{R}{r}+\mathrm{i}kZ=0.\tag{13}$$

消除 $Z(r)$ 和 $P(r)$ 得到关于 $R(r)$ 的方程：

$$\boxed{r^2\frac{\mathrm{d}^2R}{\mathrm{d}r^2}+r\frac{\mathrm{d}R}{\mathrm{d}r}-(1+(kr)^2)R=0.}\tag{14}$$

Windows 下安装 LAMMPS

发表于 2018-01-12 更新于 2019-01-23 分类于 Tutorial

Windows 下安装 LAMMPS 可以参照 LAMMPS 官网教程，本文详细介绍如何安装串行版（serial）和并行版（mpi）LAMMPS。

必要软件包

  • LAMMPS，64 位下载地址，32 位下载地址；百度云盘备用下载。
  • MPICH，64 位下载地址，32 位下载地址；百度云盘备用下载。

注意：

  • LAMMPS 从 20170127 版本之后串行版和并行版需要单独下载，其中不带-MPI是串行版，带-MPI是并行版。在此版本之前安装 LAMMPS 均会在bin目录生成lmp_serial, lmp_mpi两个文件。
  • LAMMPS 安装之后的lmp_serial, lmp_mpi均相当于绿色文件，可以复制到任意文件夹。
  • MPICH 是 Argonne 实验室开发的并行程序，Windows 下 MPICH 最高版本停留在 1.4.1p1 版，更低版本见此。
  • MPICH 更高版本已由 Argonne 实验室交给微软（MS-MPI）开发，此并行软件并不适用于 LAMMPS 计算。
  • 其它软件如 Intel Parallel Studio XE 也含有 MPI 软件，其用法与 MPICH 中 MPI 一样，此处不予介绍。
  • 以下要用到命令提示符（cmd），快捷使用请参照命令提示符（cmd）右键菜单。
